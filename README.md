A premium, bilingual digital experience designed to establish institutional authority for **Mohammed Al-Dowayesh**, Managing Partner at CMS Global. This project blends academic prestige with state-of-the-art web technology.

---

## ✨ Project Vision

A digital extension of a legal institution. Built to evoke trust, authority, and professional excellence.

### 🏆 Core Pillars
- **Uncompromising Authority**: High-end institutional aesthetics
- **Arabic-First Excellence**: Native RTL experience with seamless English LTR toggle
- **Performance & Motion**: GSAP and Lenis for smooth interactions
- **Privacy-Centric Architecture**: Secure, form-less contact model

---

## 🛠️ Technical Masterpiece

- **Frontend**: `Next.js 16.2` (App Router) with `TypeScript`
- **Styling**: `Tailwind CSS v4` with CSS variables
- **Motion Layer**: `GSAP 3.14`, `Motion`, `Lenis`
- **Internationalisation**: `next-intl` for bilingual routing
- **SEO**: Sitemap, Robots, Schema.org

---

## 🌍 Bilingual Orchestration

- **Logical Properties**: `ps-`, `pe-`, `ms-`, `me-` for RTL/LTR
- **Arabic**: Formal Modern Standard Arabic (فصحى)
- **English**: Formal British register

---

## 🔒 Security & Privacy

- **No Forms**: Direct WhatsApp/Phone CTAs only
- **Security Headers**: CSP, X-Frame-Options, HSTS
- **No External Scripts**: Except Google Fonts

## 📦 Project Architecture
```
src/
├── app/[locale]/       # Multi-language routing
├── components/
│   ├── sections/   # Hero, About, Portfolio, etc.
│   └── ui/        # Atomic components
├── lib/           # GSAP, Lenis
├── messages/       # ar.json, en.json
└── i18n/        # next-intl config
```
